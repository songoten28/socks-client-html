Socks Client

Socks Client is a free software design for Android Phone. It fully support for WiFi Repeater app

WiFi Repeater

Help

  • Connect to WiFi Repeater
    • Start to share WiFi by WiFi Repeater first and then click on to this item. It will help you connect to WiFi Repeater network and connect to Socks Server. Finally, back to your browser and enjoy
  • Connect WiFi
    • Automatically connect to WiFi without input SSID and Password using camera to scan QR code
  • Import from QR code
    • Automatically input ip address and port by QR code. QR code content like below:
       {
    "host": "123.163.119.65",
    "port": 1080
   }

    Or array

    [
  {
    "host": "46.225.240.122",
    "port": 4145,
    "name": "New Zealand",
    "flag": "nz"
  },
  {
    "host": "187.236.212.37",
    "port": 9050,
    "name": "Viet Nam",
    "flag": "vn"
  }
]

    Example:

  • Import from Server
    • Input a list socks server by URL or QR Code for URL. The URL content like below
            187.236.212.37:1080#VN
        46.225.240.122:4145#NZ

    Or

            187.236.212.37:1080
        46.225.240.122:4145
